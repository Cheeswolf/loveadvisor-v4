## 五、关键问题与修复

---

### 5.1 模块冲突问题（严重）

| 问题             | 原因    | 解决       |
| -------------- | ----- | -------- |
| streamlit 无法调用 | 文件名冲突 | 重命名文件    |
| import app错误   | 自引用   | 修正import |

---

### 5.2 环境问题

| 问题                        | 原因        | 解决               |
| ------------------------- | --------- | ---------------- |
| No module named streamlit | 多Python环境 | 使用 python -m pip |
| 命令无法识别                    | PATH问题    | 使用 python -m 启动  |

---

### 5.3 架构错误（关键）

| 问题          | 原因        | 解决    |
| ----------- | --------- | ----- |
| uvicorn启动失败 | 前后端混写     | 强制拆文件 |
| Streamlit警告 | UI代码被后端执行 | 架构隔离  |

---

### 🧠 核心教训

```text
Python 项目问题 ≈ 80% 是环境 + 结构问题
不是代码问题
```
