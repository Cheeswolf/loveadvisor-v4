## 🧩 MTU-前端修复-01

* 首页 500，前端无法启动

### 结果

```text

```

---

## 🧩 MTU-前端修复-02

**修复 useSearchParams Suspense 边界问题**

### 问题

* Next.js App Router 中直接使用 useSearchParams 报错

### 修复

* 抽离 UrlParamHandler（Client Component）
* 使用 `<Suspense>` 包裹

### 结果

```text
首页稳定渲染
```

## 🧩 MTU-输入体验增强-01

### 改动

* onDragEnter / onDragOver / onDragLeave / onDrop
* 统一 handleFiles 逻辑

### 结果

```text
支持拖拽上传截图
```

---

## 🧩 MTU-输入体验增强-02

**新增剪贴板粘贴上传能力**

### 改动

* onPaste 事件
* clipboardData.items → image/*
* getAsFile → handleFiles

### 结果

```text
支持 Ctrl+V 粘贴截图上传
```

---

## 🧩 MTU-前端联调排查-01

**排查 I1 未触发问题**

### 根因

```text
handleFiles 只更新状态，没有触发 image-to-text
```

### 修复

* useEffect 监听 screenshotFiles
* 自动调用 handleConvertToText

### 结果

```text
I1 调用链路打通
```

---

## 🧩 MTU-前端修复-03

**修复删除后重新上传不触发 I1 的问题**

### 根因

```text
convertedFileCountRef 未同步更新
```

### 修复

* 删除时同步计数
* 转换失败不更新计数

### 结果

```text
自动触发逻辑稳定
```
