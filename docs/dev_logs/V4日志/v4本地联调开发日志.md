## 四、核心问题与解决

### 1. OCR始终返回 mock

**原因：**

* 实际未命中真实服务 / 旧进程干扰

**解决：**

* 杀死旧进程
* 验证 service 与 route 一致性
* 确认真实分支执行

---

### 2. OCR 404 问题

**原因：**

* endpoint 未指向 `/chat/completions`
* 模型名过时

**解决：**

* 修正 URL 拼接
* 更新模型为可用版本

---

### 3. 结果区排版崩坏

**原因：**

* 三列布局 + 窄容器
* 中文文本无断行保护

**解决：**

* 响应式 grid
* break-words
* 移除固定高度限制

---

### 4. 页面结构不符合用户路径

**原因：**

* 使用后台式双栏布局

**解决：**

* 改为单栏任务流布局
