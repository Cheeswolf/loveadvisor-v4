## 三、今日核心实施内容

### 3.1 新增 R1 前置判断层：是否可判（is_judgable）

本日最大结构性改动，是在 `rules.py` 中新增：

```python
is_judgable(s2, s3)
```

并将 R1 主流程由：

```text
直接判断 relationship_stage
```

改为：

```text
先判断是否可判 → 不可判直接输出“无法判断” → 可判再进入阶段判断
```

该改动使 R1 从“强制分类模型”升级为“带拒判能力的规则系统”。

---

### 3.2 构建极弱互动识别机制（E 类拦截）

围绕 E1/E2/E3 的共性特征，新增两类核心判定函数：

#### ① `is_very_weak_s2`

用于识别：

* 单句 / 极短对话
* 多字段 UNKNOWN
* 浅话题 + 弱承接 + 温和语气
* key_signals 极少

#### ② `is_functional_minimal_interaction`

用于识别：

* 功能性对话（问一句答一句）
* 无任何 S3 高级关系信号
* 不形成关系语境

这两层逻辑叠加，使系统具备稳定识别：

* 信息不足场景
* 机械式互动
* 无关系语境样本

---

### 3.3 收紧“初识期兜底吸附”

在旧版本中：

```text
只要没有高级信号 → 默认归为初识期
```

这导致：

* E 类样本全部被吸入初识期

本轮调整为：

```text
无高级信号 + 极弱 S2 → 不可判
无高级信号 + 有基本互动结构 → 初识期
```

这一改动本质上引入了：

> **“是否形成关系语境”作为初识期触发条件**

---

### 3.4 interest_level 多轮精修（定点优化）

在 E 类修复完成后，对 `infer_interest_level()` 进行了多轮定点调整，围绕三类典型误差：

| 类型    | 典型样本    | 问题         |
| ----- | ------- | ---------- |
| 初识期高估 | A1      | 普通礼貌承接被判为中 |
| 暧昧期高估 | B2      | 轻度暧昧被判为高   |
| 拉扯期误判 | C1 / C3 | 中低档边界不稳定   |

优化方向包括：

* 收紧“初识期 / 中”的触发条件（必须出现升级迹象）
* 收紧“暧昧期 / 高”（需接近双向推进）
* 为“低兴趣拉扯”增加专门分支
* 引入 relationship_probe 保护机制，避免误伤 C1

---
