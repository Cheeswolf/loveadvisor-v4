# 二、三方分工（强制执行）

---

## 🟢 你（项目Owner）

### 你只负责：

1. 定阶段目标
2. 执行 Claude Code
3. 提交运行结果
4. 做最终确认

---

### ❌ 你不负责：

* 写代码
* 设计架构
* 调试细节
* 自己给 Claude Code 写任务

---

## 🔵 ChatGPT（架构总控）

### 我负责：

1. 拆解阶段任务
2. 输出 Claude Code 指令
3. 审查代码结果
4. 纠偏与优化
5. 维护整体方案一致性

---

## 🟣 Claude Code（执行层）

### 它负责：

* 写代码
* 建目录
* 修改文件
* 实现模块
* 执行任务

---

### ❌ 它禁止：

* 修改规则层（rules）
* 改项目架构
* 自行扩展功能
* 脱离任务执行

# 三、开发流程（严格执行）

---

## 标准流程（每一轮必须走）

```text
你给目标
↓
我生成任务单 + Claude Code提示词
↓
你执行Claude Code
↓
得到代码结果
↓
你发给我
↓
我验收
↓
进入下一阶段
```

---

## ❗强制规则

1. 每次只做一个阶段
2. 每次只做一个模块
3. 没验收不能进入下一步
4. 不允许直接操作 Claude Code（必须用任务单）
